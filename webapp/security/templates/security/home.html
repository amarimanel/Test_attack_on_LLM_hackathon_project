Processus d'audit

Comment ça marche ?

1. Rédigez l'Injection

Tapez votre prompt d'attaque malveillant dans le terminal sécurisé.

2. Ciblez les IA

Sélectionnez Mistral, Llama, Gemma ou GPT-2 pour le test.

3. Analysez le Score

Recevez le statut instantané : BYPASS ou PROTÉGÉ + Télémétrie.

4. Enregistrez

Sauvegardez le test pour enrichir le dataset global et le Dashboard.
